# Update vba-blocks

__Windows__

In powershell, run the following:

```shellsession
iwr https://vba-blocks.com/install.ps1 | iex
```

__Mac__

In terminal, run the following:

```shellsession
curl -fsSL https://vba-blocks.com/install.sh | sh
```
